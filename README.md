# SwiftCSharp
Comparing Swift and C#

| Variables   |      Swift     |          C#         |
|-------------|:--------------:|:-------------------:|
| boolean     |      Bool      |         bool        |
| constant    |       let      |        const        |
| declaration |       var      |         var         |
| float       |  Float, Double |    float, double    |
| integer     |       Int      |         int         |
| optional    |  ? (optional)  |     ? (nullable)    |
| tuple       |      tuple     |     System.Tuple    |
| string      | String (value) | string  (reference) |
